# Tree-Height-Measurement-Using-UWB-and-Drone
การวัดความสูงของต้นไม้ด้วยยูดับเบิลยูบีและโดรน โดยใช้ ESP32 UWB 2 ตัวทำหน้าที่เป็น tag และ anchor
- Tag ติดตั้งขึ้นไปกับโดรน
- Anchor ต่อกับ laptop ภาคพื้นดิน เพื่อเก็บค่าที่ได้และบันทึกเป็นไฟล์ CSV
- write_file.py สำหรับบันทึกค่าที่ได้จาก anchor พร้อมเวลาจริงที่ทำการบันทึก โดยบันทึกเป็นไฟล์ csv
  - สามารถเปลี่ยนตัวแปร PORT เป็น port ที่เชื่อมกับ anchor สำหรับ laptop ของผู้ใช้งาน
